# 故 차병호님 부고 (mrchafather)

김정훈의 장인 **故 차병호님**의 모바일 부고장입니다.
정적 페이지로 제작되어 GitHub Pages로 배포됩니다.

- **배포 주소**: https://dev-kjh.github.io/mrchafather/

## 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 부고장 본문 (HTML + 인라인 CSS, 외부 API 불필요) |
| `img/map.jpg` | 양평장례식장 위치 지도 이미지 |
| `CLAUDE.md` | 프로젝트 지침 및 개인정보(PII) 마스킹 정책 |

## 개인정보 처리

유족의 개인정보 보호를 위해 다음 정보는 페이지에 표시하지 않습니다.

- 유족(상주) 개인 휴대폰 번호 → 연락처는 문자·카카오톡으로 개별 안내
- 부의금 계좌 → 이 페이지에 미노출

장례식장 대표번호(031-772-1024)만 표시합니다. 자세한 정책은 [`CLAUDE.md`](./CLAUDE.md) 참고.

## GitHub Pages 활성화 방법

1. 저장소 상단 **Settings** 탭 이동
2. 좌측 메뉴 **Pages** 선택
3. **Build and deployment → Source**: `Deploy from a branch`
4. **Branch**: `main` / 폴더 `/ (root)` 선택 후 **Save**
5. 잠시 후 `https://dev-kjh.github.io/mrchafather/` 에서 확인

## 로컬 미리보기

`index.html` 파일을 브라우저로 직접 열면 됩니다.
